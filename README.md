# Implementation-of-Simple-Document-Searching

# Overview

The aim of the project was to implement a basic document search tool capable of processing documents and returning relevant results.

# Features

- Search across documents using user input
- Basic text processing and matching logic
- Structured handling of document data

- Testing and debugging of search behaviour
